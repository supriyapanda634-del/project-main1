# 📝 Personal Todo

Keeping track of daily tasks shouldn't be complicated, so I built **Personal Todo** as a simple and practical task management application. It started as a way for me to learn full-stack development, but along the way it became a project where I could apply concepts like REST APIs, MongoDB, CRUD operations, and responsive UI design.

With this app, you can create tasks, update them, mark them as completed, organize them by priority, and remove tasks you no longer need. Everything is stored in MongoDB, so your data remains available even after refreshing the page.

This project was built using the **MERN Stack** (MongoDB, Express.js, React, and Node.js) and focuses on writing clean, maintainable code while gaining hands-on experience with both frontend and backend development.

## ✨ Features

* Create, edit, and delete tasks
* Mark tasks as completed or pending
* Search and filter tasks
* Organize tasks by priority
* Responsive design for desktop and mobile
* Persistent storage with MongoDB

## 🛠️ Built With

* React
* Vite
* Node.js
* Express.js
* MongoDB
* Mongoose
* Axios

## 🚀 Why I Built This

I created this project to strengthen my understanding of full-stack web development by building a complete application from scratch. It helped me practice connecting a React frontend with an Express API, managing data in MongoDB, and implementing the core CRUD operations that are common in real-world applications.

I'm continuing to improve this project by adding new features and refining both the user experience and the codebase as I learn.
